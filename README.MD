 #Codis tools

## 说明

+ 基于 redi-go,codis-go开发的 操作codis key的工具


功能点
------------------------------------

+ key 模糊查询
+ 大key查询
+ key删除
+ 沉默key查询
+ key复制
+ 根据前缀扫描codis集群的key,并dump或删除





+codis官网：https://github.com/CodisLabs/codis
